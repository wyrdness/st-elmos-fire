## Citation Format

When citing this repository:

```
Wyrdness Project. (2026). St. Elmo's Fire. Wyrdness Paranormal Archive.
https://github.com/wyrdness/st-elmos-fire
```

When citing scientific explanation:

```
Corona discharge phenomenon creating ionization of atmospheric nitrogen
in strong electrical fields (approximately 30 kV/cm), producing characteristic
blue-violet glow. Well-established atmospheric electrical phenomenon.
```

When citing historical sources:

```
St. Elmo's Fire documented in maritime records for millennia,
named after St. Erasmus of Formia, patron saint of sailors.
```
